# Form Builder

A clean, intuitive web application for creating metadata forms. Build forms with various field types and preview how users will interact with them.

## Features

- **Multiple Field Types**: Text, Email, Number, Date, Long Text, Dropdown, and Checkbox
- **Field Configuration**: 
  - Mark fields as required
  - Allow multiple values per field
  - Custom placeholder text
  - Dropdown options management
- **Live Preview**: See exactly how your form will look to users
- **Reorder Fields**: Move fields up and down to arrange your form
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. **Add Fields**: Click on field type buttons in the left panel to add fields to your form
2. **Configure Fields**: Expand each field to set its label, placeholder, and options
3. **Toggle Options**: Mark fields as required or enable multiple values
4. **Preview**: Switch to the Preview tab to see your form in action
5. **Reorder**: Use the up/down arrows to rearrange fields

## Tech Stack

- React 19
- TypeScript
